再現！手書き立体からの擬似3D空間
========

###後期実験 OpenCV/GL 自由課題

##作品概要:
球を操って擬似３D空間を体験できる作品です。
手書きの擬似３D物体との当たり判定や、物体の裏に隠れてしまう様子などをも再現することで、あたかもそこに３D空間があるかのように表現しています。

##操作マニュアル：
SPACEキーでプレイ画面に移動。
プレイ画面中では矢印キーで移動、SPACEキーでジャンプ。
